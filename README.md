# Stitch & Spin

A colorful crochet creature challenge for inventing original amigurumi dolls. One button spins two animal wheels and two accessory wheels together, then reveals the complete mashup.

## Customize the wheels

Edit the four item lists near the top of `app/page.tsx`. Each wheel currently has ten entries with a name and emoji illustration.

## Run locally

```bash
npm install
npm run dev
```

The site deploys automatically to GitHub Pages whenever `main` is updated.
